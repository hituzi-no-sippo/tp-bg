.Embedded flag
[NOTE]
====
link:{asciidoctor_docs_url}/asciidoctor.js/latest/cli/options/#document-conversion[
Use `--embedded` flag^].

`--embedded` is used to convert only the body of the HTML.
Without `--embedded`,
the header and footer will be added and CSS will also be included.
